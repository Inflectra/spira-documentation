# Release Notes for Spira v9

## Version 9.0 (April 2026)
!!! info "Summary"
    With version 9 of Spira, we have upgraded many behind the scenes plumbing to improve stability and performance across the application. It also provides a solid foundation for future innovation.

    System administrators can now opt to enforce all users logging in with a username and password (including with LDAP) to also log in using an 2-Step Authentication token.
